# NIGHTFALL TACTICS VERSION 12 SEMI-POLISHED

This build is an actual upgraded V11 pack based on the V10 prototype.

## Added in V11
- Arcade difficulty selection: Beginner / Normal / Hard / Nightmare
- Boss floors every 5 floors with stronger scaling
- Continue system for arcade runs
- Arcade survival best-floor leaderboard saved locally
- Cleaner arcade HUD with difficulty, continues, and next-boss info
- Beginner-friendly arcade tips

## Existing features kept
- Fullscreen battle view
- Arcade survival mode
- Player vs Player
- Mana system
- Unique skills and ultimates
- Knockback and stun
- Settings menu with keybinds and audio options
- Character portraits and turntable-style preview

## Run
### Windows
Run `run.bat`

### Linux / macOS
Run `sh run.sh`

## Notes
- This is still a Java Swing prototype build.
- Assets are placeholder/prototype assets.
- Leaderboards save to local text files in the game folder.


V12 upgrades added in this build:
- responsive battle scaling for different screen sizes
- sheet-based fighter animation cycling
- upgraded combo callouts with launcher / air combo / finisher labels
- expanded stage particle effects around the Government Hall
